HTML stands for HyperText Markup Language. It is the standard language used to create webpages on the internet. This chapter will help you understand what HTML is and why it is important.

Understanding HTML

HTML is not a programming language but a markup language. It uses tags to tell the browser how to display content. For example, <h1> tag defines a main heading, and <p> tag defines a paragraph.

What are Tags?

Tags are enclosed in angle brackets <>. Most tags have an opening tag like <tagname> and a closing tag like </tagname>. The content goes between these tags.

Example of HTML:


<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

Why Learn HTML?

Because every webpage on the internet is built using HTML! Learning HTML is the first step to becoming a web developer. It helps you create websites, blogs, portfolios, and much more.
