Basic HTML Document Structure

Every HTML page has a standard structure with specific parts:

  • DOCTYPE Declaration — tells the browser which version of HTML is used
  • <html> — the root element that wraps the whole page
  • <head> — contains metadata, title, links to styles/scripts
  • <body> — contains all the visible content

Example of a basic HTML page structure:


<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Website</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is my website.</p>
  </body>
</html>

Explanation of each part:

  1. <!DOCTYPE html> – tells browser this is an HTML5 document
  2. <html lang="en"> – root element with language attribute
  3. <head> – holds page title, metadata, links to CSS & JS
  4. <body> – visible content for users
