📋 What is a Table?

Tables are used to display data in rows and columns using the <table> tag.

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>12</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>14</td>
  </tr>
</table>
Name Age
Alice 12
Bob 14

🧩 Important Tags

  • <table>: Creates the table
  • <tr>: Table row
  • <th>: Table heading cell (bold and centered)
  • <td>: Table data cell
