🌐 What are Links?

Links (or hyperlinks) are used to connect one web page to another. They use the <a> tag.

<a href="https://www.example.com">Visit Example</a>
Try Clicking Me!

🖼️ What are Images?

Images are added using the <img> tag. You need to use the src and alt attributes.

<img src="https://via.placeholder.com/200" alt="Sample Image" />

✨ Attributes You Should Know

  • href: The URL where the link goes
  • target: Where to open the link (_self, _blank)
  • src: The URL of the image
  • alt: Alternate text for the image
  • title: Tooltip text shown when hovered
