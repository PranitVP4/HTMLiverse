What Are Semantic Tags?

Semantic tags describe the meaning of the content inside them, making it easier for browsers, search engines, and humans to understand the webpage structure.

Common Semantic Tags & Examples

<header>

The <header> tag represents introductory content or navigation links.

<nav>

The <nav> tag contains the main navigation links.

<section>

The <section> tag defines a section in a document.

<article>

The <article> tag represents a self-contained composition.

<aside>

The <aside> tag is for content aside from the main content, like sidebars or callouts.

<footer>

The <footer> tag contains footer info like copyrights or contact info.
