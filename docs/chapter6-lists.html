What are Lists?

Lists allow you to present information in an ordered or unordered format.

Types of Lists

  • Ordered Lists (<ol>): Numbered lists for steps or sequences.
  • Unordered Lists (<ul>): Bulleted lists for general items.
  • Description Lists (<dl>): Lists of terms and descriptions.

Example: Ordered List

<ol>
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</ol>
  1. First step
  2. Second step
  3. Third step

Example: Unordered List

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>
  • Apple
  • Banana
  • Cherry

Example: Description List

<dl>
  <dt>HTML</dt>
  <dd>A language for creating webpages.</dd>
  <dt>CSS</dt>
  <dd>Styles the webpage for beautiful design.</dd>
</dl>
HTML
A language for creating webpages.
CSS
Styles the webpage for beautiful design.

Attributes You Should Know

  • type: Defines numbering style for ordered lists (numbers, letters, roman numerals)
  • start: Starting number for ordered list
  • reversed: Reverse order numbering
  • compact: For some browsers, makes list more compact
