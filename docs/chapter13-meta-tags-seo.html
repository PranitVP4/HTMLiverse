Chapter 13: <iframe>, <object>, <embed>

iframe object embed

1. The <iframe> Tag

The <iframe> tag is used to embed another HTML page inside the current one.

<iframe src="https://example.com" width="300" height="200"></iframe>

2. The <object> Tag

The <object> tag is used to embed media like PDFs, videos, or webpages.

<object data="sample.pdf" type="application/pdf" width="300" height="200"></object>

3. The <embed> Tag

The <embed> tag is a self-closing tag used to embed multimedia.

<embed src="video.mp4" width="300" height="200" type="video/mp4">
