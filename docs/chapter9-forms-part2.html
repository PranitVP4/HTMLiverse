📌 Let’s Level Up Our Forms!

In Part 1, we created simple forms with just text and email fields. Now, let’s add more form elements that help collect various types of data!

🔘 Radio Buttons

Radio buttons allow users to pick only one option from a list.

<label>Choose your class:</label>
<input type="radio" name="grade" value="6" /> Class 6
<input type="radio" name="grade" value="7" /> Class 7
<input type="radio" name="grade" value="8" /> Class 8

☑️ Checkboxes

Checkboxes let users select one or more options.

<label>Choose your favorite subjects:</label>
<input type="checkbox" name="subject" value="Math" /> Math
<input type="checkbox" name="subject" value="Science" /> Science
<input type="checkbox" name="subject" value="English" /> English

⬇️ Dropdown (Select)

A dropdown lets users pick one option from a menu.

<label for="favColor">Pick your favorite color:</label>
<select name="favColor" id="favColor">
  <option value="red">Red</option>
  <option value="blue">Blue</option>
  <option value="green">Green</option>
</select>

📝 Textarea

A textarea is used for longer input like comments or feedback.

<label for="feedback">Your Feedback:</label>
<textarea id="feedback" name="feedback" rows="4" cols="50"></textarea>
