✨ What is Text Formatting?

Text formatting in HTML is used to style and highlight important parts of your text content, like bold, italic, underline, etc.

🧩 Common Formatting Tags

  • <b> – Bold text (not semantic)
  • <strong> – Strongly emphasized (semantic)
  • <i> – Italic text (not semantic)
  • <em> – Emphasized text (semantic)
  • <u> – Underlined text
  • <mark> – Highlighted text
  • <small> – Smaller text
  • <del> – Deleted text
  • <ins> – Inserted text
  • <sub> – H2O (subscript)
  • <sup> – x2 (superscript)
